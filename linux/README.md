# Linux

### 这里是为Linux设计的脚本，发行版的兼容性各不一样。

**autottyd.sh**

为PVE虚拟机打开串口登录，要求硬件列表具有串口0作为入口。



**install-docker+compose**

一键安装docker和docker compose

注：需要外网环境



**jkd1.8.sh**

安装JDK 1.8环境，自测。



**macvlan-manager**

基于docker使用，创建macvlan，并且管理它们。



**menu_generator**

交互式脚本菜单创建管理器，可以协助你写脚本，完成交互式功能选择服务。



**tailscale-install**

一键安装最新的tailscale（Linux）。
